PermAudit

©derSoft 2024

This program is a simple frontend for the “find” command (or the more advanced “bfs” if you have it installed) for searching files/directories by permissions.

  • Command selection

In the top section you can select the command that you prefer, but only if the command is effectively installed. If one command isn't installed, then the selection will be disabled. If both commands aren't available, the program will exit with an error message.

  • Directory selection

In this section you can select the directory where to do the search, clicking on the little button on the right side with the folder image. You can also specify one filter to exclude one subdirectory (insert only the relative path). You can use the "*" jolly character. With the red little button on the right you can clear the "Dir excluded" field.

  • Permissions selection

Here is the fulcrum of the program. You can set the permissions filter as you wish, or negate it checking the "not" checkbox. As read in the "find" manual:
           (a) "exactly" : File's permission bits are exactly as selected
           (b) "at least" : All of the selected permission bits mode are set
           (c) "any" : Any of the selected permission bits are set

  • Second permissions selection

If you check the "and" checkbox, you can add a second filter with the same logic as the previous. It can be useful to do some advanced selection. For example you can do selections like: get all files that have at least "rw" permissions set, but not any executable permission.

  • Execute the search

Here you can select if you want to search for any type of file, just regular files or just directories. Then press "FIND".
While the program is executing the search, the led will be red.

  • Output

Here will be displayed the output of the search. You can click on the headers to sort the output. In the first column there'll be checkboxes: All the files with the checkbox checked, will be ready to have the permissions modified as selected under this section.
There's a popup menu too:
           (a) "Check all" : Check all files checkboxes
           (b) "Uncheck all" : Uncheck all files checkboxes
           (c) "Check selected" : Check all files selected with the mouse

  • Set new permissions

Here you can view the count of files that are into the output list. With the checkbox "Check all" you can check or uncheck all the files.
Then you can select the new permissions that you want to apply, and then you can press the green button at the right to start the procedure to modify the permissions. During the procedure the led will be red.
At the end of the procedure you'll see the count of modified files and the count of errors (if any).

  • Last buttons

At the bottom of the window there are three buttons:
           (a) "HELP" : It shows this help
           (b) "EXIT" : Exit from the program
           (c) "ABOUT" : It shows some info on the program and the author, and the license

NOTE :

This program save all your settings when exiting. The file will be stored under your home directory as:
"~/.config/PermAudit/settings.ini"
